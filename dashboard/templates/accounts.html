📖 Guide complet — Connecter un compte
Étape 1 — Se connecter à my.telegram.org
1 Ouvrir my.telegram.org/auth
2 Entrer votre numéro de téléphone (format international : +33612345678)
3 Telegram vous envoie un code dans l'app → l'entrer sur le site
4 Cliquer sur API development tools
Étape 2 — Créer l'application
Si vous avez déjà une app, passez directement à l'étape 3.
Remplir le formulaire "Create new application" :
App titleN'importe quel nom (ex: MyBot)
Short name5-32 caractères, lettres/chiffres (ex: mybot)
URLLaisser vide ou mettre n'importe quelle URL
PlatformDesktop ← choisir celui-ci
DescriptionN'importe quoi (ex: my app)
→ Cliquer Create application
Étape 3 — Copier les clés
Sur la page qui s'affiche, copier :
App api_id → un nombre (ex: 2042631)
App api_hash → une chaîne hex (ex: bda930c...)
⚠️ Ne partagez jamais ces clés publiquement !
Étape 4 — Connecter dans le formulaire
1 Coller API ID et API Hash dans le formulaire à gauche
2 Entrer le numéro avec indicatif pays (+33 France, +66 Thaïlande, +62 Indonésie...)
3 Cliquer Envoyer le code SMS → vous recevez un code par SMS ou dans l'app Telegram
4 Entrer le code reçu → ✅ Compte connecté !
💡 Astuce multi-comptes : Vous pouvez réutiliser les mêmes clés API (api_id/hash) pour connecter plusieurs numéros. Pas besoin de recréer une app pour chaque compte.
💡 2FA : Si votre compte a la vérification en 2 étapes activée, le système vous demandera votre mot de passe cloud après le code SMS.
