📖 Comment connecter un compte
Étape 1 — Obtenir les clés API
1 Aller sur my.telegram.org/auth
2 Se connecter avec son numéro de téléphone
3 Cliquer sur API development tools
4 Créer une app (nom = ce que vous voulez)
5 Copier l'App api_id et l'App api_hash
Étape 2 — Connecter ici
1 Coller l'API ID et API Hash dans le formulaire
2 Entrer le numéro avec indicatif (+33, +66...)
3 Cliquer Envoyer le code → recevoir le code SMS
4 Entrer le code → compte connecté !
💡 Astuce : Vous pouvez utiliser les mêmes clés API (api_id/hash) pour tous vos comptes si ils partagent la même app Telegram.
